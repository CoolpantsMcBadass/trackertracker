Cookiecutter Changelog
======================

[0.2.0] 2026-05-29
- overlay.js: floating pill injected into every page via Shadow DOM (style-isolated); shows live tracker count badge; click to expand panel with full tracker list, category badges, plain-English descriptions, and banner status; collapses on outside click or X button; receives push updates from background via TRACKER_UPDATE / BANNER_STATUS_UPDATE messages; requests initial state on load via GET_MY_TAB_DATA; respects per-site disable toggle
- background.js: added GET_MY_TAB_DATA handler (sender.tab.id context); added schedulePush() to debounce-push TRACKER_UPDATE to content scripts when new trackers are detected (400ms debounce, only on new trackers); BANNER_STATUS now also pushes BANNER_STATUS_UPDATE to overlay
- manifest.json: bumped to v0.2.0; added overlay.js to content_scripts
- data/trackers.json: expanded from 50 to 120 trackers — added Xandr/AppNexus, OpenX, Magnite/Rubicon, Index Exchange, PubMatic, Sovrn, The Trade Desk, MediaMath, Lotame, Oracle BlueKai, AdRoll, Moat, IAS, DoubleVerify, Kochava, Adjust, AppsFlyer, Branch, Conversant, GumGum, Revcontent, MGID, 33Across, SpotX, FreeWheel, ID5, LiveIntent, Audigent, Zeotap, Matomo, Woopra, Kissmetrics, Inspectlet, Clicky, StatCounter, Parse.ly, Pendo, Tealium, mParticle, PostHog, Contentsquare, AB Tasty, Dynamic Yield, Eloqua, ActiveCampaign, Mailchimp, Drip, Iterable, Emarsys, LiveChat, Freshchat, Olark, Tawk.to, Tidio, Crisp, Help Scout, YouTube Embeds, Vimeo Embeds, Twitter Widgets, Adobe Fonts, Spotify Embeds, Bugsnag, Rollbar, Dynatrace, Lucky Orange, Customer.io

[0.1.0] 2026-05-29
- Initial scaffold
- manifest.json (MV3) with webRequest + storage + tabs + scripting permissions
- background.js: service worker with webRequest interception, tracker domain matching, per-tab badge count, banner status storage, site enable/disable toggle
- content.js: cookie banner auto-decliner targeting OneTrust, Cookiebot, TrustArc, Osano, Termly, Didomi, Quantcast, Usercentrics, Consentmanager, CookieYes + generic fuzzy text matching; MutationObserver for late-loading banners
- popup/popup.html + popup.css + popup.js: dark-theme popup with tracker list (name, logo, category badge, plain-English description), banner status indicator, per-site toggle
- data/trackers.json: 50 trackers across advertising, analytics, social, marketing, support, performance categories
- icons: cookie-with-knife SVG + exported PNG at 16/48/128px
